Ultimate Media Player
Ultimate Media Player は、ブラウザ上で動作する高機能なフローティング・メディアプレイヤーです。
ローカルの動画や音声ファイルをドラッグ＆ドロップするだけで、プレイリストを作成し、デスクトップアプリのような感覚で自由自在に再生・移動・操作が可能です。

✨ 主な特徴
フローティング・ウィンドウ: プレイヤー外枠をドラッグすることで、画面内の好きな場所に配置可能。作業の邪魔にならない位置へ自由に移動できます。

ドラッグ＆ドロップ対応: 動画・音声ファイルをプレイヤーに直接ドロップするだけで、即座に読み込み・プレイリストへの追加が行えます。

高機能コントロールパネル:

10秒・1分単位のスキップ（前後）

シークバーによる直感的な再生位置調整

音量調節 & ミュート切り替え

フルスクリーン表示対応

スマートプレイリスト: 読み込んだ複数のファイルを自動でリスト化。曲が終わると次のファイルへ自動遷移します。

キーボードショートカット:

Space: 再生 / 一時停止

← / →: 10秒スキップ

🕹 使い方
起動: index.html をブラウザで開きます。

ファイルの追加: 再生したい動画や音声ファイルを、中央の「ファイルをドロップして再生」エリアへドラッグ＆ドロップします。

移動: プレイヤーの上部や余白をドラッグすると、ウィンドウを移動できます。

リスト操作: 右側のプレイリストから、再生したいファイルを直接選択できます。

🛠 技術構成
Language: HTML5, CSS3, JavaScript (Vanilla JS)

API:

Drag and Drop API: ファイル読み込み

URL.createObjectURL: ローカルリソースの一時的なURL生成

Fullscreen API: 全画面表示

Design: Segoe UI / Netflix-style Dark Mode
